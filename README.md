Project ini berisi kumpulan query SQL yang digunakan untuk menganalisis keterlambatan penerbangan menggunakan PostgreSQL. Analisis difokuskan pada pengolahan data keterlambatan keberangkatan dan kedatangan, serta identifikasi penyebab keterlambatan seperti weather delay dan late aircraft delay. Seluruh script dibuat dan dijalankan menggunakan DBeaver sebagai database client, sehingga seluruh proses analisis dilakukan sepenuhnya melalui SQL tanpa melibatkan visualisasi langsung di dalam kode.

Dalam project ini, fokus utama berada pada analisis performa maskapai dan pola keterlambatan berdasarkan kategori delay yang tersedia pada dataset. Query yang digunakan mencakup perhitungan total dan rata-rata keterlambatan, persentase ketepatan waktu (on-time performance), serta pembagian keterlambatan berdasarkan jenis penyebabnya. Analisis bandara belum dilakukan dalam project ini, dan aspek visualisasi data seperti grafik atau dashboard tidak dibuat melalui SQL. Sebagai gantinya, visualisasi dilakukan menggunakan Tableau sebagai alat pendukung untuk menyajikan hasil analisis dalam bentuk grafik yang lebih informatif dan mudah dipahami.

Project ini disusun sebagai latihan dan penguatan kemampuan dasar dalam analisis data berbasis SQL, khususnya untuk memahami bagaimana data penerbangan dapat diolah untuk menghasilkan insight terkait performa maskapai dan faktor-faktor penyebab keterlambatan. Repository ini dapat digunakan sebagai referensi bagi siapa pun yang ingin mempelajari analisis data menggunakan SQL dengan pendekatan sederhana dan fokus pada penggunaan query untuk menghasilkan insight dasar sebelum divisualisasikan melalui tools seperti Tableau.
